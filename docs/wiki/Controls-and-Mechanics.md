# Controls and Mechanics

Soft Recall Demo is currently designed for desktop browser play with mouse and keyboard.

## PC Controls

- Mouse click: inspect hotspots, select choices, and use Doorways.
- Doorways: move between accessible rooms.
- `H`: Help/controls, if supported by the current build.
- `M`: Memory Book, if supported by the current build.
- `Escape` / `Backspace`: close or go back, if supported by the current build.
- `Enter` / `Space`: confirm or advance, if supported by the current build.

## Object Hotspots

Hotspots are small glowing markers placed over objects in the first-person scene. They should remain visually aligned with the scene container and should not drift away from their clickable target.

Object hotspots are for inspection, memory fragments, routine steps, and narrative choices.

## Doorways

Doorways are the reliable movement system. When a room becomes available, it should appear in a small in-world Doorways or Available Paths control. The apartment map may also help orient the player, but it should not be the only way to navigate.

## Tutorial / Help

The title screen and any Help/controls surface should explain the minimum needed controls: inspect objects, advance VN text, open the Memory Book, and move through Doorways.

## If Stuck

- Finish any visible VN text by clicking or pressing confirm.
- Choose from any visible choice buttons.
- Use the Doorways strip to move between available rooms.
- Open the Memory Book to review what has been noticed.
- If the front door does not offer an ending yet, complete at least one kitchen routine and one bathroom routine.
