# Roadmap

## Current

PC-first stable demo.

Focus on core playability: reliable object interactions, Doorways movement, Memory Book access, ending access, build stability, and GitHub Pages compatibility.

## Next

UI/UX polish and a longer playthrough.

Improve readability, hotspot affordance, VN text rhythm, ending clarity, and overall demo pacing after core gameplay is stable.

## Later

Research-informed optional layer.

Add optional context only in docs, Learn More, Memory Book optional areas, or post-ending material.

## Deferred

Mobile/PWA.

Responsive and installable app work should wait until the desktop demo is mechanically reliable.

## Future Possibility

Controller or console-minded port.

This is conceptual only and not currently supported.

## Guiding Principle

Gameplay stability comes before expansion.
