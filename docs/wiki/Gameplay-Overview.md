# Gameplay Overview

Soft Recall Demo is a first-person visual-novel point-and-click experience set in an early-morning apartment.

## Current Playable Loop

The intended route is:

Title -> Begin -> Tutorial/Help or control hints -> Bedroom -> object interactions -> Hallway -> Kitchen/Bathroom -> Memory Book -> Front Door/Ending

The player begins in the bedroom, inspects small objects, uses phone or support cues, moves through the hallway, completes a simple routine in the kitchen and bathroom, checks the Memory Book, and reaches an ending at the front door.

## Tone

The tone should stay intimate, quiet, and slightly uncanny without becoming horror. The apartment should feel early-morning, lived-in, and modern. Memory uncertainty should be suggested through objects, pacing, support cues, and player choices rather than direct explanation.

## Narrative Rules

- Show, do not lecture.
- Keep the story grounded in ordinary objects and actions.
- Do not make clinical claims in main narration.
- Keep research and explanatory context optional and outside the main VN text.
