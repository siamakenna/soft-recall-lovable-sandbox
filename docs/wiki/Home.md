# Soft Recall Demo Wiki

Soft Recall Demo is a PC-first, desktop browser, first-person visual-novel point-and-click demo about a quiet morning, memory uncertainty, routine, and support.

The current project goal is a stable playable vertical slice: a complete short route that loads reliably, supports mouse and keyboard interaction, and can be deployed as a static GitHub Pages site.

## Project Direction

Soft Recall Demo should remain focused on playable atmosphere and core interaction stability before expanding scope. The demo is not a dashboard, clinical module, mobile app, or backend product.
