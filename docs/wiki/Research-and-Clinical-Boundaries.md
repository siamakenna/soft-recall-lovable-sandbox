# Research and Clinical Boundaries

Soft Recall Demo may be research-informed, but it is a narrative game demo, not a medical product.

## Clear Boundaries

- It is not a diagnostic tool.
- It is not a screening tool.
- It does not claim to represent one specific disease.
- It does not provide treatment or medical guidance.
- It is not medical advice.

## In-Game Tone

Main visual-novel narration should remain show-don't-tell. It should not include academic citations, clinical labels, or explanatory lectures.

## Optional Research Placement

Future research links should appear only in optional areas such as:

- documentation
- Learn More areas
- Memory Book optional context
- post-ending material

They should not interrupt main gameplay.

## Source Standards

Future sources should be limited to:

- official NIH/NLM/NIA/NINDS/.gov sources
- PubMed
- PMC
- DOI-backed peer-reviewed sources

Do not add new sources as part of this page.
