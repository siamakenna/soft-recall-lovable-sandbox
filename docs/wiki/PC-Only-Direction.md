# PC-Only Direction

Soft Recall Demo is PC-only and desktop-first for now.

The current supported target is mouse and keyboard play in a desktop browser. This direction keeps the project focused on reliable core gameplay: object inspection, VN text, room movement, support cues, Memory Book review, and ending access.

## Why Mobile Is Deferred

Mobile and PWA work are intentionally deferred. Core gameplay stability matters more than responsive expansion at this stage. Touch targets, portrait layouts, install prompts, offline behavior, and app-shell concerns should not distract from the playable vertical slice.

## Future Input Possibilities

Controller and console-minded ports are conceptual only. They may influence future design thinking, but they are not currently supported.

## Current Goal

The current goal is stable mouse and keyboard desktop play.
