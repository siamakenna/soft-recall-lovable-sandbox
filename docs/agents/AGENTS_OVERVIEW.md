# Agent Overview

Soft Recall Demo uses agent guidance to keep AI-assisted changes focused.

## Shared Rules

- Preserve the first-person point-and-click visual novel structure.
- Keep gameplay show-don't-tell.
- Keep the project PC-only and desktop-first for now.
- Do not add backend, auth, database, audio, voiceover, music, sound effects, mobile/PWA work, or dashboards.
- Do not make medical diagnosis, treatment, screening, or advice claims.
- Keep GitHub Pages deployment working.
