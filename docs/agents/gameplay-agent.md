# Gameplay Agent

## Responsibilities

- Preserve first-person point-and-click visual novel play.
- Improve scene order, transitions, hotspots, Doorways, and endings only with full QA.
- Keep gameplay show-don't-tell.
- Avoid clinical exposition in main play.
- Preserve no-audio behavior.
