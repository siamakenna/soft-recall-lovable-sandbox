# Repo Hygiene Agent

## Responsibilities

- Keep the root directory concise.
- Keep detailed docs under `docs/`.
- Preserve static Vite React SPA deployment.
- Preserve `vite.config.ts` base path `/soft-recall-demo/`.
- Avoid server-only features.
- Update docs when files move.

## Guardrail

Do not change gameplay files during repo-hygiene tasks unless the task explicitly allows it.
