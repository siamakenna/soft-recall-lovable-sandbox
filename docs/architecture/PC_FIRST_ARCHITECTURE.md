# PC-First Architecture

Soft Recall Demo is a desktop-first browser game.

## Runtime

- Vite
- React
- Static GitHub Pages deployment
- No server runtime
- No backend/auth/database

## Entry Point

The active app starts in `src/main.tsx` and renders `src/game/SoftRecall.tsx`.

## Interaction Model

- First-person scene art
- Embedded object hotspots
- Doorways navigation fallback
- Visual novel text box
- Memory Book
- Keyboard and mouse support for PC playthrough QA
