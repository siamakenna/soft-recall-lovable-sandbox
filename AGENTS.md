# AGENTS.md

Soft Recall Demo is a static Vite React single-page game. AI coding tools should keep changes focused, preserve the current first-person point-and-click visual novel structure, and avoid adding systems outside the current PC-only demo scope.

Read the full agent guidance before making changes:

- [Agent overview](docs/agents/AGENTS_OVERVIEW.md)
- [Gameplay agent](docs/agents/gameplay-agent.md)
- [Repo hygiene agent](docs/agents/repo-hygiene-agent.md)
- [QA agent](docs/agents/qa-agent.md)
- [Research boundaries agent](docs/agents/research-boundaries-agent.md)
- [Release agent](docs/agents/release-agent.md)

Core guardrails: no backend, auth, database, audio, voiceover, music, sound effects, mobile/PWA expansion, or clinical diagnosis/treatment framing. Build success is required, but manual PC playthrough QA is required for any gameplay-facing change.
